DIGITAL TEXT TOOLKIT

These projects highlight some of what is possible with my digital text toolkit; a set of HTML, CSS, and JavaScript programming methods to create meaningful and effectice engagement with digital text.
Words Birth Worlds Poem
One of my attempts to answer the question: 'how does a poem get someone out of prison?' I wrote the poetry and used JavaScript to coreograph the movement for this piece.
Thesis Presentation
My thesis presentation at NYU's Interactive Telecommunications Program. I coded my own presentation deck using Javascript, HTML, and CSS instead of using Powerpoint or Keynote which felt too restrictive.
Front end ui demo
An interactive interface for applying methods developed from my digital text toolkit to user input in real time.
